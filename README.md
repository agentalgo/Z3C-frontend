# SPA ZATCA Dashboard

An admin dashboard built with React for ZATCA (Zakat, Tax and Customs Authority) compliance and management.

### Installation:

```bash
   yarn install
```

### Run Application:

```bash
   yarn dev
```

### Production Build:

```bash
   yarn run build
```

### Resources Used:

 - **React Router Dom (ver 7.13)**
